Sample Code 1: Project 2 - Driving the Turtle

This code creates and moves a turtle.

# Trinket link here: https://trinket.io/python/37d6744782db

import turtle

my_tortoise = turtle.Turtle()   # Create a turtle
my_tortoise.color('blue')       # set the color to blue
my_tortoise.backward(50)        # draw a blue line of length 50 
my_tortoise.right(90)           # change direction to the right
my_tortoise.forward(25)         # draw a blue line of length 25               
my_tortoise.left(90)            # change direction to the left
my_tortoise.forward(50)         # draw a blue line of length 50
my_tortoise.right(90)           # change direction to the right
my_tortoise.forward(25)         # draw a blue line of length 25
my_tortoise.right(90)           # change direction to the right
my_tortoise.forward(50)         # draw a blue line of length 50
my_tortoise.right(180)          # change to opposite direction
my_tortoise.up()                # lift up the tail/pen
my_tortoise.forward(100)        # move forward 100 without drawing

my_tortoise.down()              # put the tail/pen down
my_tortoise.left(90)            # change direction to the left
my_tortoise.color('red')        # change the color to red
my_tortoise.forward(25)         # draw a red line 
my_tortoise.left(180)           # change to opposite direction
my_tortoise.forward(25)         # go over red line again
my_tortoise.left(90)            # change direction to the left
my_tortoise.forward(50)         # draw a red line of length 50
my_tortoise.left(90)            # change direction to the left
my_tortoise.forward(50)         # draw a red line of length 50
my_tortoise.left(90)            # change direction to the left
my_tortoise.forward(35)         # draw a red line of length 35
my_tortoise.right(180)          # change to opposite direction
my_tortoise.forward(70)         # go over red line again and draw another 35 segment
my_tortoise.up()                # lift up the tail/pen

Sample Code 3: Data Analysis

This code showcases data analysis techniques such as fitting data or performing statistical analysis.

# Linear regression example

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# Sample data
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 3, 5, 7, 11])

# Perform linear regression
slope, intercept, r_value, p_value, std_err = stats.linregress(x, y)

# Predicted values
y_pred = slope * x + intercept

# Plot the data and the regression line
plt.scatter(x, y, color='blue', label='Data points')
plt.plot(x, y_pred, color='red', label='Fitted line')
plt.title('Linear Regression')
plt.xlabel('X')
plt.ylabel('Y')
plt.legend()
plt.grid(True)
plt.show()
